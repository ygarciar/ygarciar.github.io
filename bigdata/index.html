Objetivos

El objetivo del curso es realizar una introducción a la programación en Python, ofreciendo al alumno una visión de las posibilidades que ofrece; por una parte la programación de aplicaciones, y por otra parte el papel fundamental que actualmente tiene Python en el contexto del analisis de datos y Big Data. Python ofrece un conjunto de módulos matemáticos y estadísticos muy útiles para análisis y representación de datos. El curso pretende ofrecer puntos de referencia a partir de los cuales se pueda profundizar tanto en el sentido de la programación como en la ciencia del análisis de datos. Para ello se espera cubrir los siguientes subobjetivos:

  • Introducir los distintos entornos de desarrollo. El entorno interactivo Jupyter para ejecutar y crear programas.
  • Conocer los elementos básicos del lenguaje.
  • Conocer y saber utilizar los módulos de análisis de datos Numpy y Pandas.
  • Importación y exportación de datasets. Limpieza y tratamiento de los datos. Agrupación de los datos.
  • Conocer el módulo matplolib para la representación de datos.

Requisitos para acceder al curso

Este curso es interesante tanto para estudiantes con pérfil tecnológico como con un pérfil menos tecnológico. Aunque no es necesario estar familiarizado con los conceptos básicos de programación, es conveniente que los alumnos tengan un conocimiento básico de informática a nivel de usuario. Python es un lenguaje ideal para introducir los principios de la computación y por tanto para iniciarse en la programación. En este sentido ofrece una oportunidad a aquellos estudiantes que sin tener nociones de informática quieren dar sus primeros pasos en la codificación de programas.

Contenidos del curso

PRIMER MÓDULO. Introducción a Python.

  1. El lenguaje de programación Python. Entornos de desarrollo: Jupyter.
  2. Elementos básicos de Python: funciones, estructuras de control, estructuras de datos, objetos, etc.
  3. Entrada/Salida en Python.
  4. Almacenamiento de datos en Python
    1. Formatos textuales: JSON, XML y HTML.
    2. Bases de datos: MongoDB

SEGUNDO MÓDULO. Herramientas de Python para análisis de datos

  1. La librería Numpy y Pandas.
  2. Visualización de datos con la librería Matplotlib.
  3. Herramientas estadísticas de Python.
  4. Limpieza, preparación, transformación y mezcla de datasets.
  5. Agregación de datos.

Metodología docente

Este curso es de carácter meramente práctico, en el que al comienzo de cada sesión se presentarán una serie de conceptos teóricos que serán ilustrados mediante la realización de programas en Python. Todas las sesiones se realizarán en una sala con ordenadores donde los estudiantes, bajo la supervisión del profesor, podrán poner en práctica los conocimientos aprendidos realizando una serie de ejercicios prácticos. La evaluación se realizará mediante la calificación de dichos ejercicios.
